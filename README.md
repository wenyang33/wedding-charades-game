# 💒 你比划我猜 - 婚礼派对游戏

一款专为婚礼现场设计的「你比划我猜」互动游戏 Web App，适合投影到大屏幕使用。

## 🎮 游戏特色

- **三队对战** — 支持 3 个队伍同时参赛，队伍名称可自由编辑
- **5 分钟倒计时** — 每队有 5 分钟比赛时间，倒计时结束自动停止
- **93 个词库** — 涵盖日常生活、动物、运动、影视人物、婚礼相关、搞笑表演等类别，随机出题不重复
- **Pass 机制** — 每队有 3 次 Pass 机会，遇到不会的词可以跳过
- **手动记分** — 主界面可输入每队分数，由裁判负责记录
- **游戏规则** — 内置游戏规则弹窗，一键查看

## 🚀 使用方法

1. 直接用浏览器打开 `index.html` 即可使用
2. 建议全屏投影到大屏幕
3. 点击队名可编辑名称
4. 点击「开始比赛」进入游戏
5. 游戏中点击「答对了」或「Pass」切换词语

## 📋 游戏规则

1. 每队派出两人比划，剩下的人猜
2. 比划者可以用肢体动作和语言描述，但**不能说出词语中的任何一个字**
3. 每队有 **5 分钟** 比赛时间
4. 每队有 **3 次 Pass** 机会
5. 猜对一个词得 1 分，Pass 不扣分
6. 三队全部完成后，得分最高的队伍获胜！

## 🛠 技术栈

- 纯 HTML + CSS + JavaScript，无需任何框架或构建工具
- 单文件部署，开箱即用
