# 💒 你比划我猜 - 婚礼派对游戏

一款专为婚礼派对设计的「你比划我猜」互动小游戏，让婚礼现场欢乐满满！

## 🌐 在线体验

👉 **https://wenyang33.github.io/wedding-charades-game/**

## 🎮 游戏简介

这是一个基于网页的多人派对游戏，支持三支队伍轮流比赛。比划者用肢体语言表演词语，队友负责猜测，在限定时间内猜对越多越好！

## ✨ 功能特点

- 🎯 **三队对战** — 支持三支队伍轮流比赛，可自定义队名
- ⏱️ **2分钟倒计时** — 每队有2分钟比赛时间，最后30秒自动预警
- ⏭️ **Pass 机制** — 每队3次 Pass 机会，策略性跳过难题
- 📖 **内置规则说明** — 随时查看游戏规则
- 🎲 **随机词库** — 90+ 精选词汇，涵盖多种类别，所有队伍共享词池（用过不重复）
- ⏸️ **暂停功能** — 支持随时暂停和继续
- 💖 **婚礼主题** — 粉色浪漫风格，带有心形飘落动画
- 📱 **响应式设计** — 适配手机和平板等移动设备

## 📋 游戏规则

1. **游戏形式**：每队派出两人比划，剩下的人猜。比划者可以说简短提示（如"三个字"），之后只能用肢体语言
2. **时间限制**：每队有 2 分钟比赛时间
3. **Pass 机会**：每队 3 次 Pass，用完后只能继续猜
4. **答对规则**：猜对后点击"答对了"进入下一题
5. **犯规说明**：比划者说出词语中的字视为犯规，该词不得分
6. **计分方式**：每答对一词得 1 分，由裁判在主界面记录得分
7. **获胜条件**：三队完成后，最多的获 3 分，第二名 2 分，第三名 1 分

## 🚀 如何使用

1. 直接用浏览器打开 `index.html` 即可运行，无需安装任何依赖
2. 点击队伍名称可自定义队名
3. 点击「开始比赛」进入该队的游戏环节
4. 游戏结束后由裁判手动输入各队得分
